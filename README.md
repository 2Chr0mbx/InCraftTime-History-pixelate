# InCraftTime-History
___

## 导语 (Intro)

### 1- 这是啥 (What is it)

如仓库名所述，InCraftTime-History 致力于收集 `InCraftTime Server` 的历史资料(~~当然是黑料合集~~) , 以便任何 好奇服务器历史 \ 想要黑料取证 \ ~~单纯吃瓜~~ 的玩家随时调取查阅.

### 2- 什么是ICT (What is ICT)

ICT,全程InCraftTime,是一个自 *2019年运营至今*[^1] 的**Minecraft服务器**,其长期在 `BiliBili` 视频平台宣传,并以公益服[^2]作为服务器特色.[^3]

其服务器服主正版游戏ID:MineSunshineone ,玩家一般称其为"小麦".

### 3- 为什么会有 "InCraftTime-History" (Why) 

自2025年11月以来,仓库作者认为,关于ICT服务器的诸多历史遗留问题仍未得到解决,并且服主对服务器近期[^1]发生事情消极处理并且进行"捂嘴".本人认为有关ICT服务器历史资料应**去中心化保存**,此份文件为Github备份,并会视情况持续更新或修改.

作者将2020年至2025年[^4]零散资料整理,汇总,并建里此仓库.向后世玩家提供参考资料.

<u>**历史不应被遗忘**</u>

### 4- 一些其他内容 (Extra)

本仓库在互联网上的发布版本应为 `pixelate` 版本,为保护部分相关人员,故对部分内容进行**打码遮盖处理**(包括不限于原链接,头像,昵称,~~真实姓名~~).原始版本仅应部分人(或仅作者)可见.

你可以随意下载,拷贝,引用此仓库的所有内容.

文档(*部分*)内容不一定具有时效性,且可能存在**争议性**内容,***理性讨论,全面看待!***

如有更多信息,欢迎Issue补充
